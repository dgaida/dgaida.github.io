## Academic Document Generator

</p>

Das Github Repository [**Academic Document Generator**](https://github.com/dgaida/colloquium-protocol-creator) ist ein vielseitiges Software-Werkzeug zur **automatisierten Erstellung** offizieller akademischer Dokumente an der TH Köln. Mithilfe von LLMs verwandelt das Tool PDF-Annotationen in professionelle LaTeX-Protokolle, Gutachten für Projektarbeiten oder konstruktive Peer-Review-Kommentare. Zu den Kernfunktionen gehören die automatische Extraktion von Studierenden-Metadaten aus Projekt- und Abschlussarbeiten, das **Ausfüllen von Prüfungsformularen** sowie die Integration von Outlook für den Mailversand. Ein spezielles Modul ermöglicht zudem die strukturtreue **Übersetzung von LaTeX-Klausuren** vom Deutschen ins Englische unter Beibehaltung mathematischer Formeln. Die Anwendung unterstützt verschiedene LLM-Schnittstellen wie OpenAI, Gemini oder lokale Lösungen (Ollama) und vereinfacht so den administrativen Korrekturaufwand für Prüfende erheblich.
